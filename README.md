# Controle de Envase

Sistema desenvolvido em C# (.NET 8) para controle de movimentações de tanques e cálculos de envase.

## Funcionalidades
- Controle de volume iniciado
- Transferências T1 e T2
- Cálculo de perdas e purgas
- Volume final
- Volume envasado


## Tecnologias
- C# / Windows Forms
- .NET 8.0
- Visual Studio 2022
